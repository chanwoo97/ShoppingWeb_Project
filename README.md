# 🛍️ShoppingWeb_Project

## ✨ 주요 기능 (Features)
- 회원 관리 : 회원가입, 로그인, 로그아웃, 마이페이지 기능
- 상품 : 상품 목록 조회, 상세 페이지, 카테고리별 분류
- 주문 : 장바구니 , 주문하기, 주문 내역 조회
- 결제 : 미구현
- 관리자 페이지 : 상품 등록 및 관리, 주문 관리, 회원 관리

## 🛠️ 사용 기술 (Tech Stack)
- Backend : Java, Spring Boot , Spring Security, JPA
- Frontend : HTML , CSS , JS , Thymeleaf
- Database : MariaDB
- Build Tool : Gradle
- Server : TomCat Server
